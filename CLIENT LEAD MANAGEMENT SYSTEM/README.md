# 🧾 Client Lead Management System (Mini CRM)

A simple **Client Lead Management System (Mini CRM)** built using **HTML, CSS, and JavaScript**.  
This project helps manage client leads generated from website contact forms.

---

## 🚀 Features

- 🔐 Admin Login (Username & Password)
- ➕ Add New Leads (Name, Email, Status, Notes)
- 📋 View Leads in Table Format
- ✏️ Edit Leads
- 🗑️ Delete Leads
- 🔍 Search Leads (Name / Email)
- 📊 Dashboard Analytics (Total / New / Contacted / Converted)
- 📝 Notes for each lead

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- LocalStorage (for login session)

---

---

## 📊 Dashboard Features

- Total Leads Counter  
- New Leads Counter  
- Contacted Leads Counter  
- Converted Leads Counter  

---

## 🎯 How It Works

1. Open `login.html`
2. Enter admin credentials
3. Access CRM Dashboard
4. Add / Edit / Delete leads
5. Search and track lead status
